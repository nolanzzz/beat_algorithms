## Tree (BinaryTree)
- Represent hierarchical data
- Databases (indexing)
- Autocompletion
- Compilers
- Compression (JPEG, MP3)

### Tree Traversals

- **Breadth First / Level Order**
- **Depth First**
  - Pre-order: _Root_, Left, Right
  - In-order: Left, _Root_, Right (ascending order)
  - Post-order: Left, Right, _Root_(leaves first, then root)

**Height:** bottom-up

 **Depth:** top-down
